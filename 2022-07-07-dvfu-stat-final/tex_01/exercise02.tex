ожиданием \(\mathbb{E}(X_i) = 5 a + 1\).

По выборке из 100 наблюдений оказалось, что \(\bar X = 16\).

Найдите оценку \(\hat a\) методом моментов.
\begin{answerlist}
  \item 3.6
  \item 3.2
  \item 2.8
  \item 3
  \item 3.4
  \item нет верного ответа
\end{answerlist}
\end{question}
