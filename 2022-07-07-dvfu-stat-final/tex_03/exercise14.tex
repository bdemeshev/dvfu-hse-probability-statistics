
\begin{question}

ожиданием и дисперсией \(5\). Вектор \((X, Y)\) имеет многомерное
нормальное распределение с корреляцией \(0.5\).

Найдите \(\mathrm{Var}(Y\mid X = 2)\).
\begin{answerlist}
  \item \(3.75\)
  \item \(0.5\)
  \item \(0.75\)
  \item нет верного ответа
  \item \(1\)
  \item \(5\)
\end{answerlist}
\end{question}
